Mashweb.Club: Home of end-user-programmed web apps

{{< blocks/link-down color="info" >}}
{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Mashweb.Club presents a radically new approach to the creation of websites and especially web applications meant to be usable by a very broad class of people. Mashweb.Club links to live prototypes of web-page-editor interfaces to demonstrate that it could be possible for end users to build their own web pages from scratch and decide every detail of their web pages' look and structure simply, easily, and quickly, without any prior learning. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="Documentation consolidated!" %}} The GitBook and GitHub Pages sites have been combined and converted to a Hugo website using the Docsy theme and served by CloudFlare Pages.
